# LumitySpeaker
Bot I made in less than 24 hours for my friend and I.

# How to use
Install the bot using this <a href="https://discord.com/oauth2/authorize?client_id=1436831494768300253">link</a> for discord. I should be hosting this bot, but if not then you can make another version and run that instead.

### **/help** to view commands in discord.

### **/github** brings you to this page. This is to view the source code and commands more in detail.

### **/translate** translate a message from lumity speak or english lettering.
The way **/translate** works is that it reads the *first* letter of the message.
*It checks the **first** word for an emoji, and translates it to english if so, and vice-versa*

### **/translatetolumity** translate a message to lumity, from english lettering
Does not check first letter of word for emoji. Straight to lumity code.

### **/translatetoenglish** translate a message to english letters, from lumity code.
Does not check first letter of word for emoji. Straight to english letters.

### **/dictionary** tells you the matchup of letters to emojis.

  Be careful. Make sure you're installing py-cord using pip. Not pycord. Pycord is a seperate libary from py-cord.
